    Project Architecture

    1 Document Ingestion

    PDF, PPTX, and text files are extracted and split into searchable chunks.

    2 Vector Memory

    Chunks are embedded with Gemini and stored in ChromaDB with source metadata.

    3 Agent Router

    LangGraph retrieves local evidence, then routes to Gemini, OpenAI, or Groq generation.

    4 Report Generator

    Answers can be returned in the UI or exported as PPT, PDF, and Excel reports.
